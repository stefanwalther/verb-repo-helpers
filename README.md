> Commonly used helpers for adding repository-related data to templates in verb projects.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install verb-repo-helpers --save
```

## Usage

```js
var helpers = require('verb-repo-helpers');
```
